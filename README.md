# K-Beauty SKU Launch Validator

Validate consumer demand before mass production and reduce the risk of overproduced beauty SKUs.

미션: K-beauty 브랜드가 대량 생산 전에 실제 소비자 반응을 검증하고 더 정확한 SKU 출시 결정을 내릴 수 있도록 지원합니다.

비전: 수요에 기반한 생산 결정을 장려하여 과잉 생산을 줄이고 검증된 수요가 있는 제품만 생산되는 뷰티 산업에 기여합니다.

## 서비스 소개
이 도구는 신제품을 대량 생산하기 전에 다음을 빠르게 검증할 수 있도록 설계되었습니다.

- 어떤 포지셔닝 메시지가 더 반응이 좋은지
- 예상 가격대가 초기 관심을 끌 수 있는지
- 소비자 행동 신호(CTA 클릭, 대기자 등록, 샘플 요청 등)가 충분한지

## 주요 기능
- 랜딩 페이지: 문제/해결/결과를 한눈에 보여주는 B2B SaaS 스타일 소개
- SKU 테스트 빌더: 브랜드/제품/타겟/성분/가격/메시지/CTA 입력
- 가상 제품 페이지 미리보기: 메시지 A/B/C 반응(좋아요) 및 CTA 상호작용 수집
- 테스트 대시보드:
  - 총 페이지뷰, CTA 클릭, 클릭률
  - 대기자 등록 수, 전환율
  - 메시지 A/B/C 선호도
  - 우승 메시지, 가격 인사이트, 수요 점수, 출시 권고안
- 방문자 시뮬레이션:
  - "Simulate 100 U.S. Visitors" 버튼으로 로컬 규칙 기반 행동 시뮬레이션 실행
- 리포트:
  - 종합 수요 점수
  - 최고 성과 메시지
  - 가격 민감도 인사이트
  - 다음 액션 제안

## 기술 스택
- React
- Vite
- TypeScript
- Tailwind CSS
- 로컬 상태(Local State) 기반 로직
  - 백엔드 없음
  - 데이터베이스 없음
  - 인증 없음
  - 외부 API 없음

## 실행 방법
```bash
npm install
npm run dev
```

## 사용 흐름
1. 랜딩 페이지에서 테스트 시작 또는 샘플 데이터 로드
2. SKU 입력 폼 작성 후 가상 제품 페이지 생성
3. CTA 클릭/이메일 제출/메시지 좋아요로 반응 신호 수집
4. 100명 방문자 시뮬레이션 실행
5. 대시보드와 리포트에서 수요 점수 및 권고안 확인

## 샘플 테스트 데이터
"Load Sample PDRN Cream Test" 버튼을 누르면 데모 입력값이 자동으로 채워져 즉시 테스트할 수 있습니다.

## 중요 안내
이 MVP는 초기 수요 신호를 측정하기 위한 도구입니다.
실제 판매 성과를 보장하지 않습니다.
